### Проєкт 1: Онлайн-платформа Studiora
**Опис:** Платформа для персоналізованого онлайн-навчання студентів і школярів. Дає можливість проходити курси, виконувати завдання, отримувати рекомендації від наставників, використовує гейміфікацію та аналітику прогресу.
**Технології:** 
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
